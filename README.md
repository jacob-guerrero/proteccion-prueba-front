# Cliente Web - Motor de Priorización

- **Live Demo:** [[Vercel URL](https://proteccion-prueba-front.vercel.app/)]
- **Backend Repo:** [[GitHub Backend Link](https://github.com/jacob-guerrero/proteccion-prueba)]

## Descripción
Interfaz gráfica moderna para la gestión y visualización de solicitudes priorizadas. Construida para consumir la API REST de Spring Boot.

## Stack Tecnológico
- **Core:** React 18 + Vite.
- **Estilos:** Tailwind CSS.
- **Conectividad:** Fetch API nativa.
- **Despliegue:** Vercel (CI/CD automático).

## Características
- **Formulario de Ingreso:** Validación de input básica y selectores de tipo.
- **Visualización en Tiempo Real:** Lista ordenada automáticamente según la lógica de negocio del servidor.
- **Formato Inteligente:** Conversión de fechas a formato local legible.

## Ejecución Local
1. Clonar el repositorio.
2. `npm install`
3. `npm run dev`
